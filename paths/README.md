# Paths

Ordered learning sequences. A path **links** to courses and units — it does not copy their files.

| Kind | Examples |
|------|----------|
| `professional_certificate` | Coursera IBM RAG |
| `skill_track` | DataCamp Developing AI Applications |
| `career_track` | DataCamp career paths |
| `custom` | Interview prep, personal study plans |

Register in `catalog/paths.yaml`. Landing page: `index.html` (use `shared/templates/path_index_template.html`).
