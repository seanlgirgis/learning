# StudyBubble v1.0 Release Package

This is the first usable StudyBubble release package of standalone visual study-map HTML files.

## How To Open
- Open `index.html` first, or open any topic `.html` file directly from File Explorer.
- Recommended start file: `case_capacity_overview.html`.

## Direct-Open Behavior
- Files are designed for direct open (`file://`) with no local server required.
- Topic data is embedded in each generated HTML.

## Included
- `case_capacity_overview.html`
- `case_capacity_evidence.html`
- `case_capacity_feature_table.html`
- `case_capacity_forecast.html`
- `case_capacity_decision.html`
- `python_overview.html`
- `pandas.html`
- `assets/feature-table-example.svg`

## Not Included
- No server
- No backend
- No database
- No npm
- No React

## Known Accepted Warning
- Browser console may show a `file://` unique-origin warning during direct-open usage.

## Best Demo Path
- `case_capacity_overview.html`
  - -> Evidence and Features
  - -> Feature Table Deep Dive
  - -> back to Evidence and Features
  - -> back to Case Overview
