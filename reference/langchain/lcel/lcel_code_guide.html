Core pattern

concept

prompt | llm | StrOutputParser()

Build objects once. Pass a dict to chain.invoke() — keys match {holes} in the template.

prompt = PromptTemplate.from_template("Tell me a {adjective} joke about {content}.")
chain = prompt | llm | StrOutputParser()
chain.invoke({"adjective": "funny", "content": "chickens"})

Components

component

PromptTemplate

invoke(dict) in chains; format(**dict) for quick tests.

component

StrOutputParser

Normalize output to plain str.

component

RunnableLambda

Optional wrapper for .format(); prompt | llm is enough.

Traps

trap

format vs invoke

Chains use invoke(dict).

trap

Chain ≠ agent

Fixed steps vs tool-picking. SQL lab does not run queries.

trap

Template wins

Role/tone in template beats constraints in the user question.

Two styles

Direct (default)

chain = prompt | llm | StrOutputParser()

Notebook / Coursera

chain = RunnableLambda(format_prompt) | llm | StrOutputParser()
