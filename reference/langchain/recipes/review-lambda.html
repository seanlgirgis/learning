Recipe · module 05

Product review analysis

Structured sentiment + features + summary — Coursera style with RunnableLambda.

Problem

Run the same analysis template over a batch of review strings. Output format is fixed in the prompt.

Ingredients

01 02 03 04 05 RunnableLambda

Setup

from langchain_core.prompts import PromptTemplate
from langchain_core.output_parsers import StrOutputParser
from langchain_core.runnables import RunnableLambda
from watson_llm import make_watsonx_llm, GenParams

llm = make_watsonx_llm({
    GenParams.MAX_NEW_TOKENS: 512,
    GenParams.TEMPERATURE: 0.2,
})

Recipe code (Coursera pattern)

template = """
Analyze the following product review:
"{review}"

Provide your analysis in the following format:
- Sentiment: (positive, negative, or neutral)
- Key Features Mentioned: (list the product features mentioned)
- Summary: (one-sentence summary)
"""
product_review_prompt = PromptTemplate.from_template(template)

def format_review_prompt(variables):
    return product_review_prompt.format(**variables)

review_analysis_chain = (
    RunnableLambda(format_review_prompt)
    | llm
    | StrOutputParser()
)

reviews = [
    "I love this smartphone! The camera quality is exceptional...",
    "This laptop is terrible. It's slow, crashes frequently...",
]

for i, review in enumerate(reviews):
    print(f"==== Review #{i + 1} ====")
    print(review_analysis_chain.invoke({"review": review}))
    print()

Direct equivalent (your pattern)

review_analysis_chain = product_review_prompt | llm | StrOutputParser()
# same invoke: {"review": review}

reviewOne product review string
Trap: Model may add extra sections (e.g. Recommendation). Tighten the template or lower temperature if format drifts.
