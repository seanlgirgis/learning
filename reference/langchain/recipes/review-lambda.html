Product review analysis

Batch of reviews in → structured sentiment + features + summary out. Coursera pattern: RunnableLambda(format_review_prompt) | llm | StrOutputParser().

1 — Problem

E-commerce team has many product review strings; you need the same analysis format on each: sentiment, features mentioned, one-line summary. A plain loop over invoke is enough — no separate prompt file per review.

Not this recipe: JSON schema enforcement with parsers (Module 11), or the simpler prompt | llm one-liner (equivalent here — lambda is for learning the Runnable step).

4 — Script

from langchain_core.prompts import PromptTemplate
from langchain_core.output_parsers import StrOutputParser
from langchain_core.runnables import RunnableLambda
from watson_llm import make_watsonx_llm, GenParams

llm = make_watsonx_llm({
    GenParams.MAX_NEW_TOKENS: 512,
    GenParams.TEMPERATURE: 0.2,
})

template = """
Analyze the following product review:
"{review}"

Provide your analysis in the following format:
- Sentiment: (positive, negative, or neutral)
- Key Features Mentioned: (list the product features mentioned)
- Summary: (one-sentence summary)
"""
product_review_prompt = PromptTemplate.from_template(template)

def format_review_prompt(variables):
    return product_review_prompt.format(**variables)

review_analysis_chain = (
    RunnableLambda(format_review_prompt)
    | llm
    | StrOutputParser()
)

reviews = [
    "I love this smartphone! The camera quality is exceptional and the battery lasts all day. The only downside is that it heats up a bit during gaming.",
    "This laptop is terrible. It's slow, crashes frequently, and the keyboard stopped working after just two months. Customer service was unhelpful.",
]

for i, review in enumerate(reviews):
    print(f"==== Review #{i + 1} ====")
    print(review_analysis_chain.invoke({"review": review}))
    print()

Traps

  • Lambda returns PromptTemplate object — Must return formatted str; LLM expects text in this chain.
  • Skipping **variables — format(review=…) breaks when you add more holes later.
  • Thinking lambda is required — prompt | llm does the same job here; lambda teaches the explicit formatting step.

Rebuild from memory

Close this page. Write the script from these steps only:

  1. Import PromptTemplate, StrOutputParser, RunnableLambda, make_watsonx_llm, GenParams.
  2. llm = make_watsonx_llm({MAX_NEW_TOKENS: 512, TEMPERATURE: 0.2}).
  3. Build product_review_prompt with {review} and three bullet format lines.
  4. Define format_review_prompt(variables) returning product_review_prompt.format(**variables).
  5. Chain RunnableLambda(format_review_prompt) | llm | StrOutputParser(); loop reviews; invoke and print.
