String PromptTemplate + pipe

Lab 19 — {adjective} and {topic} holes filled from a dict, then sent through prompt | llm.

1 — Problem

You want a reusable joke prompt with two variable slots — adjective and topic — without f-strings scattered through the code. LangChain’s string PromptTemplate plus LCEL pipe is the minimal pattern before chat templates.

Not this recipe: single-hole summarize (Lab 09), role-based ChatPromptTemplate (Lab 20), or raw string invoke (Lab 16).

2 — Prerequisites

NeedModuleCheck

3 — Wiring diagram

{"adjective": "funny", "topic": "cats"}
       → PromptTemplate("Tell me one {adjective} joke about {topic}")
       → make_watsonx_llm({MAX_NEW_TOKENS: 80})
       → str printed

4 — Script

from langchain_core.prompts import PromptTemplate
from watson_llm import GenParams, make_watsonx_llm

prompt = PromptTemplate.from_template("Tell me one {adjective} joke about {topic}")
input_ = {"adjective": "funny", "topic": "cats"}

llama_llm = make_watsonx_llm({GenParams.MAX_NEW_TOKENS: 80})
chain = prompt | llama_llm

print(chain.invoke(input_))

Playground: 19.params.py

Notebook step: print(prompt.invoke(input_)) — see filled string before the LLM runs.

5 — Critical lines

6 — Variants

  • Preview formatted prompt — prompt.invoke(input_) before adding | llm.
  • Add parser — Append StrOutputParser() for stable str → summarize recipe.
  • Lower temperature — make_watsonx_llm({GenParams.TEMPERATURE: 0.1, …}) → temperature compare.
  • Chat roles — Swap to ChatPromptTemplate → Lab 20.

7 — Debug

Traps

  • make_llm() — Playground uses make_watsonx_llm() only.
  • prompt.format(…) only — Formats text but never calls the model. Need chain.invoke.
  • Wrong hole name — Notebook uses {topic}; older drafts used {content}. Match the template exactly.

Rebuild from memory

Close this page. Write the script from these steps only:

  1. Import PromptTemplate, GenParams, make_watsonx_llm.
  2. PromptTemplate.from_template with {adjective} and {topic}.
  3. Build input_ = {"adjective": "funny", "topic": "cats"}.
  4. llama_llm = make_watsonx_llm({GenParams.MAX_NEW_TOKENS: 80}).
  5. Wire chain = prompt | llama_llm and print(chain.invoke(input_)).
