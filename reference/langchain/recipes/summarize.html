Recipe

Summarize in one sentence

Long text in → short summary out. One template variable: content.

Problem

Turn a long passage into a single-sentence summary without hand-writing the prompt each time.

Ingredients

01 PromptTemplate 02 WatsonxLLM 03 pipe 04 parser

Setup (every LCEL recipe)

from langchain_core.prompts import PromptTemplate
from langchain_core.output_parsers import StrOutputParser
from watson_llm import make_watsonx_llm

llm = make_watsonx_llm()  # WatsonxLLM from WATSONX_* env

Run set_env.ps1 first. Infra only: LLM plug-in · IBM SDK: model.generate (not for pipes).

Recipe code

passage = """
The rapid advancement of technology in the 21st century has transformed
healthcare, education, and transportation. AI, ML, and IoT are changing
how we solve everyday and complex problems.
"""

template = "Summarize the {content} in one sentence."
prompt = PromptTemplate.from_template(template)

summarize_chain = prompt | llm | StrOutputParser()

print(summarize_chain.invoke({"content": passage}))

invoke keys

KeyValue
contentFull text to summarize

Flow

{"content": passage}
  → prompt fills template
  → llm generates one sentence
  → StrOutputParser → str
Trap: Use chain.invoke({"content": passage}) — not prompt.format() alone. Coursera sometimes wraps format in RunnableLambda; direct prompt | llm is equivalent.
