Recipe

Generate SQL text

English description → SQL string. No database connection.

Problem

Write a query from a natural-language spec. Output is text only — you run it elsewhere (or not at all).

Ingredients

01 02 03 04

Setup

from langchain_core.prompts import PromptTemplate
from langchain_core.output_parsers import StrOutputParser
from watson_llm import make_watsonx_llm

llm = make_watsonx_llm()

Recipe code

description = """
Retrieve the names and email addresses of all customers from the
'customers' table who have made a purchase in the last 30 days.
The table 'purchases' contains a column 'purchase_date'.
"""

template = """
Generate an SQL query only with no explanation or commentary based on the {description}

SQL Query:
"""
prompt = PromptTemplate.from_template(template)
sql_generation_chain = prompt | llm | StrOutputParser()

print(sql_generation_chain.invoke({"description": description}))

invoke keys

KeyValue
descriptionTables, columns, filters in plain English
Not agentic: No DB tool. Fixed chain ≠ agent that queries Italy’s population from a live database.
