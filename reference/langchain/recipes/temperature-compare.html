Compare temperature 0.8 vs 0.1

Lab 18 — two make_watsonx_llm() instances, same model, different GenParams. Params are fixed at build time.

1 — Problem

You need to see how sampling temperature changes output style — creative vs precise — on the same watsonx model. Tweaking one LLM object between calls does not work; params bind when you construct the Runnable.

Not this recipe: prompt templates (Lab 19), chat roles (Lab 17), or Coursera’s Granite-vs-Llama notebook exercise.

2 — Prerequisites

NeedModuleCheck
make_watsonx_llm()07Lab 16 invoke works
GenParams.TEMPERATURE090.8 vs 0.1 passed in dict
Same WATSONX_MODEL_ID02Both LLMs read same env model
MAX_NEW_TOKENS cap09Keeps compare output readable

3 — Wiring diagram

prompt str
       ├→ make_watsonx_llm({TEMPERATURE: 0.8, MAX_NEW_TOKENS: 80}) → creative
       └→ make_watsonx_llm({TEMPERATURE: 0.1, MAX_NEW_TOKENS: 80}) → precise
       → same prompt → two different str outputs

4 — Script

import os
from watson_llm import GenParams, make_watsonx_llm

print("Model:", os.environ["WATSONX_MODEL_ID"])

llm_creative = make_watsonx_llm({GenParams.TEMPERATURE: 0.8, GenParams.MAX_NEW_TOKENS: 80})
llm_precise = make_watsonx_llm({GenParams.TEMPERATURE: 0.1, GenParams.MAX_NEW_TOKENS: 80})

prompts = [
    "Write a short poem about artificial intelligence",
    "What are the key components of a neural network?",
    "List 5 tips for effective time management",
]

for prompt in prompts:
    print("=" * 60)
    print("PROMPT:", prompt)
    print("--- creative (0.8) ---")
    print(llm_creative.invoke(prompt))
    print("--- precise (0.1) ---")
    print(llm_precise.invoke(prompt))
    print()

6 — Variants

  • In a pipe — Pass overrides when building the LLM used in prompt | llm → string template (Lab 19).
  • Stricter summarize — make_watsonx_llm({GenParams.TEMPERATURE: 0.1}) on summarize chain.
  • Other params — Explore TOP_P, TOP_K in Module 09.

Traps

  • Tweaking params between calls on one LLM — make_watsonx_llm() freezes params at construction. Make a new instance per setting.
  • Expecting Coursera’s two-model exercise — Local lab compares two temperatures on one env model, not Granite vs Llama.
  • Using make_llm() — Use make_watsonx_llm(); the old name does not exist.
