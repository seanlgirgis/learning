Lab 17

Chat messages invoke

SystemMessage + HumanMessage + optional AIMessage → llama_llm.invoke([...])

Setup

from langchain_helper import make_llm, GenParams
from langchain_core.messages import HumanMessage, SystemMessage, AIMessage

llama_llm = make_llm({GenParams.MAX_NEW_TOKENS: 80})

Example 1 — system + human

msg = llama_llm.invoke([
    SystemMessage(content="Book recommender. One short sentence."),
    HumanMessage(content="I enjoy mystery novels, what should I read?"),
])
print(msg)

Example 2 — include AIMessage (fake prior turn)

msg = llama_llm.invoke([
    SystemMessage(content="Fitness bot. One short sentence."),
    HumanMessage(content="I like HIIT, what should I do?"),
    AIMessage(content="Try a CrossFit class."),
    HumanMessage(content="How often should I attend?"),
])
print(msg)

Example 3 — human only

print(llama_llm.invoke([
    HumanMessage(content="What month follows June?"),
]))
Not memory: each call is fresh unless you put history in the list. For pipes with roles use Lab 20.
