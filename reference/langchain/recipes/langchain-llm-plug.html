Plug in WatsonxLLM for LangChain

Helper builds WatsonxLLM from env — use in pipes, chains, and invoke.

Problem

You need an llm object for prompt | llm | parser without repeating credentials in every script or notebook cell.

Ingredients

02 WatsonxLLM 07 watson_llm 03 pipe

Playground lab (Lab 16)

from langchain_helper import make_llm

llama_llm = make_llm()
print(llama_llm.invoke("Who is man's best friend?"))

Playground — params override

from langchain_helper import make_llm, GenParams

llm = make_llm()

# One-shot string (no pipe)
print(llm.invoke("Who is man's best friend?"))

# Override temperature (Exercise 1 style)
llm_creative = make_llm({GenParams.TEMPERATURE: 0.8})
llm_precise = make_llm({GenParams.TEMPERATURE: 0.1})

Jupyter notebook

from coursera_llm_model import make_llm

llama_llm = make_llm()  # WATSONX_* env; name kept for later cells

print(llama_llm.invoke("Who is man's best friend?"))

llama_llm is still your WATSONX_MODEL_ID — not Coursera’s Llama unless you set that model in env.

Plug into a pipe

from langchain_core.prompts import PromptTemplate
from langchain_core.output_parsers import StrOutputParser
from langchain_helper import make_llm

llm = make_llm()
prompt = PromptTemplate.from_template("Tell me a {adjective} joke about {content}.")
chain = prompt | llm | StrOutputParser()

print(chain.invoke({"adjective": "funny", "content": "chickens"}))
