Module 06 · IBM SDK

ModelInference

IBM’s low-level watsonx client — not a LangChain object. Dict in, dict out.

When to use

Course demos that call model.generate(...). Quick “is watsonx wired up?” checks. You want the raw API response (IBM JSON shape).

Do not use this inside prompt | ??? | parser — use Module 02 WatsonxLLM instead.

Object

NamePython typeRole
modelibm_watsonx_ai.foundation_models.ModelInferenceCall watsonx text generation API directly

How you get it (helpers)

WhereImport
Playground labfrom watson_helper import model, model_id, credentials
Jupyter notebookfrom coursera_watsonx_model import model, model_id, credentials

Both read the same WATSONX_* env vars from set_env.ps1.

Main method

msg = model.generate("In today's sales meeting, we ")
text = msg["results"][0]["generated_text"]
print(text)
PartMeaning
msgFull IBM response (dict)
msg["results"]List of completions (usually one)
["generated_text"]The model’s continuation as a string
