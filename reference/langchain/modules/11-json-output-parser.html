Module 11 · Lab 21

JsonOutputParser

Structured output — data class defines shape; parser builds prompt instructions and parses reply to dict.

Benefit (one idea)

Automation recipe: define a Pydantic class once → same object drives get_format_instructions() and end-of-pipe parsing. No hand-written JSON rules every time.

Objects

StepCode
Shapeclass Joke(BaseModel): setup, punchline
Parserjoke_parser = JsonOutputParser(pydantic_object=Joke)
Instructionsjoke_parser.get_format_instructions() → partial in prompt
Chainprompt | llm | joke_parser → dict
