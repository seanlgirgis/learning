Chat message types

Labeled messages in a list — not one plain string. Use with WatsonxLLM.invoke([...]).

When to use

Notebook “Chat message” section: give the model roles (system rules, user question, fake prior AI reply). Not memory — you pass the full list each call.

Objects

TypeRole
SystemMessageInstructions / persona
HumanMessageUser text
AIMessagePrior assistant line you supply (for fake history)
from langchain_core.messages import HumanMessage, SystemMessage, AIMessage

Call pattern

llama_llm = make_llm()

msg = llama_llm.invoke([
    SystemMessage(content="You are a book recommender. One short sentence."),
    HumanMessage(content="I enjoy mystery novels, what should I read?"),
])
print(msg)  # str
