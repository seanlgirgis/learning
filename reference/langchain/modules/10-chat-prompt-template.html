ChatPromptTemplate

String template → chat template: list of role messages, pipe-friendly.

When to use

After Module 01 (PromptTemplate = one string). Use when you want system + user tuples and LCEL prompt | llm.

Pattern A — roles + variable

from langchain_core.prompts import ChatPromptTemplate

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    ("user", "Tell me a joke about {topic}"),
])
print(prompt.invoke({"topic": "cats"}))  # message list

chain = prompt | llama_llm
print(chain.invoke({"topic": "cats"}))

Pattern B — MessagesPlaceholder

Slot in a list of messages at invoke time (e.g. user-supplied history chunk).

from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder
from langchain_core.messages import HumanMessage

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    MessagesPlaceholder("msgs"),
])
prompt.invoke({"msgs": [HumanMessage(content="What is the day after Tuesday?")]})

vs Module 08

Module 08: you build HumanMessage lists by hand. Module 10: template + dict fills roles and placeholders — better for reusable pipes.
