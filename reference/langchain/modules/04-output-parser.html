Module 04

StrOutputParser

Step 3 — always give you plain text from the model step.

When to use

End of pipe when you need a str. Required when using ChatWatsonx (message objects). Optional with WatsonxLLM (often already a string).

Object

ObjectImport
StrOutputParser()from langchain_core.output_parsers import StrOutputParser

Usage

from langchain_core.output_parsers import StrOutputParser

parser = StrOutputParser()
chain = prompt | llm | parser

Job (one line)

Normalize whatever the model returned → str you can print or pass on.

With WatsonxLLM today: output may look the same with or without parser — keep it in the pipe for consistency when you switch to chat models.

Also

Plain str out (this module) vs structured dict / list — see Module 11 — JsonOutputParser.
