Module 01

PromptTemplate

Mail-merge for prompts — fills {holes} before the model runs.

When to use

Any time the prompt has variable slots (topic, review, question, …). In LCEL, put prompt as step 1 in the pipe.

Object

ObjectTypeRole
promptPromptTemplateFormatter Runnable — not the model

Methods (we use)

PromptTemplate.from_template(template)

Build from a string with {variable} placeholders.

template = "Summarize the {content} in one sentence."
prompt = PromptTemplate.from_template(template)

prompt.invoke({"key": value})

Use in chains — same interface as other Runnables.

prompt.invoke({"content": "Long paragraph here..."})

prompt.format(key=value, ...)

Quick test only — returns plain str, not for full pipelines.

prompt.format(content="Short text")

Properties / rules

NameMeaning
{variable} in templateMust match keys in invoke({...})
Return of invokePromptValue (chain handles it)
Return of formatPlain str
Trap: format vs invoke — chains use chain.invoke(dict), not format.
