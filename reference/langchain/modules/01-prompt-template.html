PromptTemplate

Mail-merge for prompts — fills {holes} before the model runs. Step 1 in almost every LCEL pipe.

Job (help-desk story)

Blank form with labeled fields. Customer data (content, question, review) slots in; the specialist (LLM) only sees the finished paragraph.

Problem without it

  • F-strings and .format() scattered in code — change the prompt in five places.
  • No shared contract for what variables a chain expects at invoke time.
  • Hard to inject parser instructions ({format_instructions} in Lab 21) without a template object.

Object lifecycle

  1. Build — PromptTemplate.from_template("...{var}...") or constructor with input_variables + partial_variables.
  2. Configure — Template string defines holes; partial_variables pre-fills slots (e.g. JSON format instructions).
  3. Invoke — In a pipe: parent passes dict. Standalone: prompt.invoke({"var": value}) → PromptValue.
  4. Output — PromptValue (chain feeds to LLM). Manual format() → plain str for quick tests only.

Properties

Methods (we use)

PromptTemplate.from_template(template)

Fastest path — LangChain infers holes from {braces}.

prompt = PromptTemplate.from_template("Summarize the {content} in one sentence.")

Constructor + partials (Lab 21 pattern)

prompt = PromptTemplate(
    template="Answer the user query.\n{format_instructions}\n{query}\n",
    input_variables=["query"],
    partial_variables={"format_instructions": joke_parser.get_format_instructions()},
)

prompt.invoke({"key": value})

Runnable interface — what LCEL pipes call internally.

Imports (Coursera → your venv)

from langchain.prompts import PromptTemplate

Your playground (1.x)

from langchain_core.prompts import PromptTemplate

Traps (from your labs)

  • format() in production path — Labs 08–14 work with chain.invoke(dict). Using only format() skips the LLM step mentally and breaks when you add parsers.
  • Hole name mismatch — Template says {content} but invoke passes {"text": ...} → fails before watsonx is called.
  • Literal braces — JSON examples in the template need doubled braces {{ }} or LangChain treats them as holes.

Recall drills

What does prompt.invoke({"content": "hi"}) return vs prompt.format(content="hi")?
invoke → PromptValue (for chains). format → str (for quick inspection).
Why does Lab 21 use partial_variables for format_instructions?
Parser generates a long instruction block once; it is not user input at invoke time — only query or subject changes per call.
Where does PromptTemplate sit in prompt | llm | parser?
Step 1 — receives the invoke dict, outputs filled prompt for the LLM step.
