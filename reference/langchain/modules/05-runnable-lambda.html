RunnableLambda

Wrap any Python function as a pipe step — Coursera uses it for .format().

When to use

Custom logic between steps. For plain template fill-in, prompt | llm is enough — lambda is optional.

Object & import

from langchain_core.runnables import RunnableLambda

Coursera pattern

def format_prompt(variables):
    return prompt.format(**variables)

chain = RunnableLambda(format_prompt) | llm | StrOutputParser()
chain.invoke({"content": passage})

variables = the dict from invoke. **variables unpacks to keyword args for format.

Direct equivalent (preferred)

chain = prompt | llm | StrOutputParser()
chain.invoke({"content": passage})
Trap: RunnableLambda(prompt) is wrong — pass a function, not a PromptTemplate object.

When lambda is worth it

Strip text, rename keys, merge fields — any Python before the model sees the string.
