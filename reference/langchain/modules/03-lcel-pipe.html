Module 03

LCEL pipe

Wire Runnables with | — fixed steps, one invoke runs them all.

When to use

Default pattern for LangChain apps: format → generate → (optional) parse.

Objects

ObjectTypeRole
promptPromptTemplateStep 1 — format
llmWatsonxLLMStep 2 — generate
parserStrOutputParserStep 3 — normalize (optional with text LLM)
chainRunnableSequenceWired pipeline

Syntax

chain = prompt | llm | StrOutputParser()

result = chain.invoke({"adjective": "funny", "content": "chickens"})

Methods

CallInputOutput
chain.invoke(dict)Dict — keys match template holesFinal step result (usually str)

Flow

dict in
  → prompt fills template
  → llm calls watsonx
  → parser (if present) → str out
Trap: Chain ≠ agent. Steps are fixed at build time — model does not pick tools.
