Module 09 · Lab 18

Generation parameters

Change behavior by building a new make_llm(params) — not between invokes on one object.

Rule

Params are fixed at make_llm() / make_watsonx_llm() time. For creative vs precise, create two LLMs:

llm_creative = make_llm({GenParams.TEMPERATURE: 0.8})
llm_precise  = make_llm({GenParams.TEMPERATURE: 0.1})

Knobs we use

ParamEffect
TEMPERATURE0.1 tight · 0.8 more varied
MAX_NEW_TOKENSCap answer length
TOP_K / TOP_PSampling pool — TOP_K: 1 can mute temperature

Exercise 1 (local)

Same WATSONX_MODEL_ID, compare 0.8 vs 0.1 on three prompt types (poem, facts, list). Document when temperature visibly matters.

Observation: With default TOP_K: 1, creative and precise may look identical — loosen TOP_K / TOP_P to see temperature bite.
