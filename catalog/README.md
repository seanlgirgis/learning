# Catalog

Machine-readable registry. Paths and plans reference IDs here — not folder paths directly.

| File | Contents |
|------|----------|
| `courses.yaml` | Coursera (`crs_`) and DataCamp courses |
| `units.yaml` | Skills, interview prep, ad-hoc learning (non-course) |
| `paths.yaml` | Tracks, certs, custom sequences — ordered member lists |

Member `kind` in paths: `course` or `unit`.
