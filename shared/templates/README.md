# HTML Templates

Copied from StudyBook `Course_starter` (canonical source). Use these when creating new content.

| Template | Use for |
|----------|---------|
| `course_index_template.html` | Coursera / DataCamp course landing (`index.html`) |
| `field_guide_template.html` | Whole-unit or whole-course field guide |
| `section_field_guide_template.html` | Chapter / module field guide |
| `sql_quick_lookup_template.html` | Quick lookup pages |
| `path_index_template.html` | Paths (tracks, certs, custom sequences) |
| `unit_index_template.html` | Skills, interview prep, non-course units |

**Rules:** Keep `index.html` as the active landing filename. Use relative links. No `_final` / `_updated` suffixes.
