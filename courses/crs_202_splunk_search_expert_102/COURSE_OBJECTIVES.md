# Course Objectives - Splunk Search Expert 102

## Main objectives

- Filter data using time modifiers and time commands.
- Use formatting functions to accommodate various time formats.
- Calculate statistics using transforming commands.
- Use mathematical and statistical eval functions.
- Compare, manipulate, and normalize data using eval and related commands.
- Enrich results with lookups.
- Use subsearches to correlate and filter data from multiple sources.

## Skills focus

- Time filtering
- Time commands
- Transforming commands
- Statistics
- eval
- Lookups
- Subsearches
- Data manipulation
- Data normalization

## Our project deliverable

By the end of this course, Sean should be able to transform Splunk search results into useful statistics, manipulate fields with eval, work with time, and enrich/correlate searches using lookups and subsearches.
