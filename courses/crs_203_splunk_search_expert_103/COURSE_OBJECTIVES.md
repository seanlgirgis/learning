# Course Objectives - Splunk Search Expert 103

## Main objectives

- Calculate co-occurrence between fields.
- Analyze data from multiple datasets.
- Understand how components of a search are processed across the Splunk data pipeline.
- Troubleshoot searches.
- Manipulate, evaluate, and analyze multivalue data.
- Create multivalue fields using multivalue eval commands and functions.
- Improve search performance using search best practices.
- Accelerate reports and use data models to optimize frequently accessed data.

## Skills focus

- Co-occurrence analysis
- Multiple datasets
- Search pipeline
- Search troubleshooting
- Multivalue fields
- Search optimization
- Report acceleration
- Data models
- Performance tuning

## Our project deliverable

By the end of this course, Sean should understand how Splunk processes searches, how to troubleshoot slow or incorrect searches, and how to use advanced SPL patterns for performance and multivalue data.
