# Course Objectives - Splunk Search Expert 101

## Main objectives

- Investigate data using Splunk Search Processing Language (SPL).
- Create searches, reports, dashboards, and visualizations.
- Use temporary and persistent fields in searches.
- Enrich data with fields from other data sources.
- Create, manage, and schedule reports and alerts.
- Use visualizations to create charts, graphs, and maps.
- Use visual formatting commands to curate dashboard appearance.

## Skills focus

- SPL basics
- Searching
- Fields
- Reports
- Alerts
- Dashboards
- Visualizations
- Incident response basics
- Ad hoc reporting

## Our project deliverable

By the end of this course, Sean should be able to write basic SPL searches, create simple reports, understand fields, and explain the difference between searching, reporting, alerting, and dashboarding.
