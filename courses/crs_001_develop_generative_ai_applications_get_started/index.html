Chapter Guides

Module 1 — Generative AI, Prompt Engineering, and Prompt Templates

Generative versus discriminative AI, foundation models, LLMs, prompting techniques, few-shot learning, and reusable prompt templates.

Module 2 — Introduction to LangChain in Generative AI Applications

LangChain components, LCEL, runnables, parallel processing, parsers, structured output, provider switching, and conversation history.

Module 3 — Build a Generative AI Application with LangChain

Model selection, evaluation, multimodel design, Flask integration, HTTP handling, and prototype-to-production concerns.

LangChain Code Patterns

Reusable imports, constructors, invocation patterns, LCEL skeletons, output parsing, and history patterns based on the code that ran locally.
