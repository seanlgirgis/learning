LCEL spine

Module 03 · Labs 08.lcel.1.py–14 · Summarize recipe.

from watson_llm import make_watsonx_llm
from langchain_core.prompts import PromptTemplate
from langchain_core.output_parsers import StrOutputParser

llm = make_watsonx_llm()
prompt = PromptTemplate.from_template("Summarize {content} in one sentence.")
chain = prompt | llm | StrOutputParser()
chain.invoke({"content": passage})
Memory rule: prompt.invoke → prepare · llm.invoke → generate · chain.invoke → run all steps.

Structured output

Module 11 · Labs 21–22.

parser = JsonOutputParser(pydantic_object=Joke)
prompt = PromptTemplate(
    template="...\n{format_instructions}\n{query}",
    partial_variables={"format_instructions": parser.get_format_instructions()},
)
chain = prompt | make_watsonx_llm() | parser  # → dict

RAG pipeline

RAG guide · Modules 12–16 · Labs 23–31.

load → split → embed → Chroma → retriever → RetrievalQA
chunks = splitter.split_documents(loader.load())
store = Chroma.from_documents(chunks, make_watsonx_embeddings())
qa = RetrievalQA.from_chain_type(llm=make_watsonx_llm(), retriever=store.as_retriever(), chain_type="stuff")
qa.invoke("what is this paper discussing?")
