# LlamaIndex Module 3 Practice Pack

This pack is designed for learning the basic RAG mental model:

Source files → Loader → Documents → Chunks/Nodes → Embeddings → Index → Retriever → Query Engine → LLM Answer

## What this pack gives you

- Strong sample documents, not random filler
- A tiny company knowledge base
- Questions that test retrieval quality
- Starter Python code for LlamaIndex
- A runbook to guide your experiments

## Suggested learning order

1. Read `runbook.md`
2. Inspect the files under `data/company_knowledge`
3. Run `starter_code/01_load_documents.py`
4. Run `starter_code/02_build_index_query.py`
5. Ask the questions in `questions/practice_questions.md`

## Important

The documents are fictional and safe for practice.
They are written to behave like real business documents: policies, runbooks, FAQs, incident notes, and product notes.
