# Internal FAQ

## Which product should I use for document question answering?

Use DocuPilot.

## Which product should I use for dashboard-style reports?

Use InsightBoard.

## Which product should I use for monitoring operational metrics?

Use SignalWatch.

## Can I upload restricted data into an AI tool?

No. Restricted data must not be pasted or uploaded into any AI tool.

## What should I do if I am unsure whether a document is confidential or restricted?

Remove sensitive sections or ask the security team before uploading it.

## What is the fastest escalation path for Sev-1?

Page the on-call engineer immediately and open an incident bridge within 15 minutes.
