# Product Notes

## InsightBoard

InsightBoard is best for dashboard-style reporting.

It supports charts, tables, filters, scheduled email reports, and manager summary pages.

InsightBoard is not optimized for long document question-answering.

## DocuPilot

DocuPilot is best for document-grounded question answering.

It supports PDF ingestion, markdown notes, source citations, and searchable policy answers.

DocuPilot is not intended to replace dashboards.

## SignalWatch

SignalWatch is best for operational monitoring.

It supports alert rules, metric thresholds, anomaly notes, and incident timelines.

SignalWatch is not intended for executive slide reporting.
