Main example

from pathlib import Path from llama_index.core import SimpleDirectoryReader DATA_DIR = Path(__file__).resolve().parent.parent / "data" / "company_knowledge" documents = SimpleDirectoryReader(str(DATA_DIR)).load_data() print(f"Loaded {len(documents)} documents") for doc in documents: print(doc.metadata.get("file_name"), len(doc.text))
Output: each item has .text and .metadata.

a) Basic directory

documents = SimpleDirectoryReader(str(DATA_DIR)).load_data()

One folder. Good for flat corpora like the practice pack.

b) Recursive directory

documents = SimpleDirectoryReader( str(DATA_DIR), recursive=True, ).load_data()

Includes subfolders.

c) Filter by extension

documents = SimpleDirectoryReader( str(DATA_DIR), recursive=True, required_exts=[".md", ".csv"], ).load_data()
Quiz pattern: recursive=True + required_exts.

d) Exclude archives / temp

documents = SimpleDirectoryReader( str(DATA_DIR), recursive=True, exclude=["*.zip", "*.tmp", "archive"], ).load_data()

e) Single or selected files

documents = SimpleDirectoryReader( input_files=[str(DATA_DIR / "01_company_overview.md")] ).load_data() paths = [str(DATA_DIR / "02_support_policy.md"), str(DATA_DIR / "07_faq.md")] documents = SimpleDirectoryReader(input_files=paths).load_data()

f) Manual in-memory Document

from llama_index.core import Document documents = [ Document(text="Policy text here.", metadata={"source": "manual"}), ]

No loader — you already have the text.

Metadata keys (file loaders)

  • file_path
  • file_name
  • file_type
  • file_size
  • creation_date
  • last_modified_date
Inspect doc.metadata once per new source type.

Safe display fallback

source = ( doc.metadata.get("file_name") or doc.metadata.get("file_path") or "unknown" )

Use when logging — not every loader fills every key.
