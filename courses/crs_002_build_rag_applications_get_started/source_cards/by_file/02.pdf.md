## Task 1 — Getting started

### Install

```bash
pip install gradio
```

### First app — greet + slider

```python
import gradio as gr

def greet(name, intensity):
    return "Hello, " + name + "!" * int(intensity)

demo = gr.Interface(
    fn=greet,
    inputs=["text", "slider"],
    outputs=["text"],
)
demo.launch(server_name="127.0.0.1", server_port=7860)
```

**Try it:** name in left box → drag slider → Submit → greeting on right.

**Why `int(intensity)`?** Slider may hand you a float; `!` repeat count must be int.

**String shortcuts:** `"text"` and `"slider"` are valid instead of `gr.Textbox()` / `gr.Slider()`.

**Local mirror:** `12_m2_gradio_greet_slider.py`

## Understanding `gr.Interface` (critical for quiz)

Designed for functions with **one or more inputs** and **one or more outputs**.

### Three core arguments

| Arg | Rule |
|-----|------|
| `fn` | Any Python function to wrap |
| `inputs` | Component count **must match** `fn` parameter count; **order matches** |
| `outputs` | Component count **must match** return values (or one component for single return) |

- **30+** built-in components: `gr.Textbox()`, `gr.Image()`, `gr.HTML()`, …
- Multiple returns → list of output components

## Task 2 — Image classification (ResNet-18)

### Step 1 — Load model

```python
import torch
model = torch.hub.load('pytorch/vision:v0.6.0', 'resnet18', pretrained=True).eval()
```

### Step 2 — `predict(inp)` function

| Step inside `predict` | Why |
|------------------------|-----|
| `inp` is PIL image | Matches `gr.Image(type="pil")` |
| `transform` → tensor | ResNet needs normalized tensor |
| `model(inp)` | Forward pass |
| `softmax` on logits | Convert to probabilities summing to 1 |
| Return `dict` | Keys = class labels, values = confidence floats |

### Step 3 — Gradio wrapper

```python
gr.Interface(
    fn=predict,
    inputs=gr.Image(type="pil"),
    outputs=gr.Label(num_top_classes=3),
    examples=["/path/lion.jpg", "/path/cheetah.jpg"],
).launch()
```

| Parameter | Meaning |
|-----------|---------|
| `gr.Label(num_top_classes=3)` | Show top 3 classes only (ImageNet has 1000) |
| `examples=[...]` | Prefilled demo rows — replace with **your** image paths |
| `share=True` in `launch()` | Temporary public URL |

**Quiz tie-in:** `Label` + `num_top_classes` appear in `09.pdf` cheat sheet.

You do **not** need to run ResNet locally — know the **input/output pairing**: Image in → Label dict out.
