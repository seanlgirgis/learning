## Interface class — three core arguments

| Term | Definition |
|------|------------|
| **fn** | Function to wrap. Each **parameter** = one input component. Return = single value or **tuple** (each tuple element = one output component). |
| **inputs** | Component(s) for inputs. **Count = parameter count.** |
| **outputs** | Component(s) for outputs. **Count = return count.** |

---

## Common input types (verbatim for quiz)

| Term | Definition |
|------|------------|
| **checkbox** | True or False **only** |
| **checkboxGroup** | **Multiple** values from predefined checkbox list |
| **dropdown** | Dropdown list; default **one** value; `multiselect=True` → one or more |
| **file** | User uploads a file |
| **image** | Select or upload image |
| **radio** | Forces user to choose **one** value |
| **slider** | Value between min and max; `value` = default; `step` = increment; **min, max, and step as integers → integer values** (cheat sheet text; lab adds **value** too — quiz uses all four) |
| **textbox** | Expandable text |

---

## Common output types

| Term | Definition |
|------|------------|
| **textbox** | Expandable text (LLM answers) |
| **label** | Classification — classes + probabilities; **`num_top_classes`** limits how many classes shown |

## Launching

- `launch()` — starts web server serving the interface
- `share=True` — temporary **public URL**; compute still runs **locally**
